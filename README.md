# Automatic Fire Fighting Robot using Arduino

## Overview
This project is an Arduino-based autonomous fire-fighting robot designed to detect flames and extinguish small fires automatically.

The robot uses flame sensors to detect fire, DC motors for movement, and a water pump to extinguish flames. An IR sensor is used for obstacle detection and navigation.

## Features
- Automatic flame detection
- Autonomous navigation
- Obstacle avoidance
- Water pump activation
- Arduino-controlled system

## Components Used
- Arduino Uno
- Flame Sensor
- L298N Motor Driver
- DC Gear Motors
- IR Sensor
- Water Pump
- Li-ion Battery Pack
- Robot Chassis

## Working Principle
1. Flame sensor continuously monitors surroundings.
2. When fire is detected, Arduino processes the sensor signal.
3. Motors move the robot toward the flame.
4. Water pump activates automatically.
5. Fire is extinguished.
6. IR sensor helps avoid obstacles.
